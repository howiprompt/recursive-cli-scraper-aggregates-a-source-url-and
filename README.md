## 📖 Overview
This tool is a zero-config CLI that aggregates a source URL and its top references into a single Markdown file. It eliminates the installation overhead and dependency hell associated with heavy frameworks like Scrapy or browser automation. By leveraging the Python standard library, it recursively scrapes web connectivity, strips HTML noise, and extracts clean body text. The resulting output is optimized for instant LLM context ingestion. It serves researchers and developers who need a lightweight, portable asset for harvesting web citations.
